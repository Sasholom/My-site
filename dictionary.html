HTML-словарик

<html>
это главный тег в котором находится вся информация страницы
<!DOCTYPE>
Позволяет понимать браузеру, что за тип страницы
<head>
служебная информация о странице
<title>
заголовок страницы
<main>
основное содержание
<header>
шапка сайта
<footer>
подвал сайта
<section>
смысловой раздел
<nav>
основная навигация
<article>
независимый раздел
<aside>
дополнительное содержание
<h1-h6>
заголовки в HTML
<p>
параграф
<meta>
кодировка страницы, ключевые слова
<ul>
неупорядоченный список
<ol>
упорядоченный список
<start>
можно поменять это стартовое число
<reversed>
меняет направление нумерации на противоположный
<dl>
обозначает сам список описаний
<dt>
обозначает термин
<dd>
обозначает описание или определение
<pre>
преформатированный текст
<code>
фрагмент кода
<q>
встроенная цитата
<blockquote>
отдельная цитата
<br>
перенос строки
<sup>
отображает текст в виде верхнего индекса
<sub>
отображает текст в виде нижнего индекса
<time>
дата и время
<em>
определяет текст, на который сделан особый акцент
<i>
применяется для обозначения текста, который отличается от окружающего текста, но не является более важным
<strong>
указывает на важность отмеченного текста
<b>
предназначен для выделения текста с целью привлечения к нему внимания, но без придания ему особой важности
<del>
выделяет текст, который был удалён в новой версии документа
<ins>
выделяет текст, который был добавлен в новой версии документа
<div>
обычно используется для группировки крупных элементов, например, нескольких абзацев, или в качестве контейнера для создания сеток страниц
<span>
используется для выделения мелких текстовых элементов: частей слов, отдельных слов или фраз, состоящих из нескольких слов
<a>
создание ссылок
<img>
изображение
<width>
ширина изображения
<heigh>
высота изображения
<alt>
Альтернативный текст изображения
<figure>
Используется для группирования любых элементов
<figcaption>
Содержит описание для тега<figure>
